Fixing the Velocity bug

Right now, if we jump too high in our game, our player will fall stright though the floor. This is because PixelPad's frame rate cannot handle speeds that are too fast. So we need to put a cap on our maximum velocity!
Luckily, this is really easy to do! Add these lines to the bottom of the scripts

obj_player / start 

        ...
        max_vel = 8


obj_player / loop 

        ...
        if y_vel > max_vel:
            y_vel = max_vel


Done! Now on to the next topic: Finding sprites!

Finding some sprites!

So our game is shaping up, but we will need a few more sprites to really bring this game together! We need:

  1. Some type of collectible
  2. Some type of enemy

For the collectible, the classic item is a coin, but you can get a lot more creative than that! Maybe a mushroom, maybe a sticker, maybe pacman for all we know?! It's your game! Just remember two things:

  1. The picture has to be a .png file
  2. The picture's background should be transparent

Once you have the images, upload them into PixelPad (using the green +) and create objects for them and add them to the game. In case you forget how to do that, please refer to the 10 billion other objects we've made.

And also, just in case here's the code!

obj_<name> / start 

        # Bind the sprite to the object
        sprite = sprite_new('spr_<name>')


game / start 

        # Show the object on screen
        object_name = object_new('obj_<name>')

        # Move the object around
        object_name.x = 100
        object_name.y = -100

Add Pac dots

We've finished our maze, so now we need to add some pac dots!

Remember that in order for our maze to fit on the srcreen, we actually shrunk the maze parts by 10%. If you didn't do that, here is the code in the game script!

game / start 

        TOP = 220
        LEFT = -200
        tsize = 14.4
        
        
        maze = object_new('obj_maze')
        maze.y = 10000
        
        
        def renderRow(rowList, rowNum):
          if len(rowList) != 28:
            print("Row", rowNum, "Does not have 28 sprites, has ", len(rowList))
            return
          
          for i in range(len(rowList)):
            if rowList[i] == -1:
              continue
            t = game.maze.create_tile(rowList[i])
            # Shrink maze by 10%
            t.sprite_height = 0.9
            t.sprite_width = 0.9
            t.x = game.LEFT + i*game.tsize
            t.y = game.TOP - rowNum*game.tsize
        
        
        room_set('rm_lvl1')
        score = 0


Ensuring that code is there is key for the pac dots to work out properly.
